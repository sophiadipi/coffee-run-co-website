# coffee-run-co-website
A responsive website developed for a local coffee shop.

## Live Demo
https://coffeerunco.ct.ws/

## Features
- Responsive design (mobile, desktop, tablet)
- Modern UI/UX layout
- Clean, user-friendl navigation

## Technologies used
-  HTML5
-  CSS3
- JavaScript

## Purpose
This project was created to design a website for a local business without an existing online presence.
